# GHL Guide — Courses: Upsells

**Where in GHL:** Memberships → Offers → select offer → Upsell tab

---

## Overview

GHL Memberships supports two upsell types:
1. **One-Click Upsell** — post-purchase offer shown after checkout; customer accepts with one click using stored payment method
2. **In-App Upsell** — locked offer cards shown on the My Library page; members can purchase additional products without leaving the platform

---

## One-Click Upsell

A one-click upsell appears immediately after a customer completes their initial purchase. The customer can accept the additional offer with a single click — no re-entering payment details.

**Where to configure:** Memberships → Offers → select offer → three-dot menu → Edit → **Upsell tab**

### Setup Steps

1. Memberships → Offers → locate the base offer → three-dot menu → **Edit**
2. Open the **Upsell tab**
3. Click **+ Add Upsell**
4. In the upsell type modal, select **One-Click Upsell** → click **Continue**
5. Choose the upsell offer from the dropdown → click **Add Upsell Offer**
6. Customize the upsell page using three tabs:
   - **Details:** headline, subtitle, fonts, colors, alignment
   - **Body:** promotional video and poster image
   - **Checkout:** button text, colors, pricing display, button styling
7. Click **Save** → toggle Status to **Active**

**Payment methods supported:** Stripe and PayPal (for one-click transactions)

**Sequential upsells:** Multiple upsells can be chained — each displays after the previous one is accepted or declined.

**Performance tracking:** Monitor views, clicks, and conversions in the Offer analytics section.

**To edit or remove:** Upsell tab → three-dot menu on the upsell → Edit or Delete

---

## In-App Upsell

An in-app upsell displays locked offer cards inside the member's My Library page. Members see what they don't have and can purchase it without leaving the portal.

**Where to configure:** Memberships → Offers → select offer → three-dot menu → Edit → **Upsell tab**

### Setup Steps

1. Memberships → Offers → locate the offer → three-dot menu → **Edit**
2. Open the **Upsell tab** → click **+ Add Upsell**
3. Select **In-App Upsell** → click **Continue**
4. Choose the offer to display on the library page from the dropdown
5. Customize using three tabs:
   - **Details:** title, subtitle, alignment, fonts, banner background, button styling
   - **Course Body:** description, curriculum details, font treatments
   - **Instructor:** fonts and colors for instructor information
6. Click **Preview** to review the layout
7. Toggle Status to **Active** → **Save**

**Key behaviors:**
- Already-purchased offers are NOT shown as locked in My Library
- Multiple in-app upsells can be attached to one offer
- All configured checkout payment methods work with in-app upsells

## Upsell Analytics

**Where:** Memberships → Offers → select offer → Analytics section

Metrics tracked per upsell:
- Views
- Clicks
- Conversions

Use these to identify which upsells drive the most incremental revenue and which need headline or copy improvements.
